# HomeLab Dotfiles & Configuration

## Overview
A comprehensive setup for a home lab environment with optimized configurations for various services and applications.

## Key Components

### Desktop Environment
- **Hyprland**: Modern Wayland compositor with custom keyboard shortcuts and window management
- **Waybar & Mako**: System status bar and desktop environment components
- **Hyprpaper**: Background wallpaper management with custom themes

### Docker Services
- **Pi-hole**: DNS filtering and ad-blocking for network devices
- **Jellyfin**: Personal media server for streaming movies, music, and shows
- **Caddy**: Web server with SSL/TLS capabilities for hosting websites and services
- **Netdata**: System monitoring dashboard for real-time performance tracking
- **Glances**: System monitoring with resource usage visualization

## Configuration Highlights
- Monitor setup with dual displays (2560x1440 and 1920x1080)
- Custom keyboard bindings for common tasks (application launch, window management, workspace switching)
- Dynamic window layouts with split view functionality
- Automated startup scripts for key applications

## Usage
This repository contains configuration files for all primary services in the home lab, ensuring consistent and optimized performance across all components. All configurations are designed to work seamlessly together, providing a fully functional and customizable home lab environment.

> Note: This setup is optimized for a home lab with dual monitors, personal media streaming, and network-level ad-blocking.
